# Kashvi Creation - Exclusive Designer Sarees

## Overview

Kashvi Creation is a full-stack e-commerce platform designed for an exclusive designer saree store in Surat. The website provides a seamless experience for both customers and administrators, making it easy for a local store to manage online sales and customer interactions effectively.

The platform includes an admin dashboard and a client-side interface, offering features such as product filters, search and sorting, multi-image and video uploads, a contact form for inquiries, a blog section, real-time order tracking, and customer reviews. The website also features a personalized chatbot designed specifically for Kashvi Creation to assist customers.

## Features

### Customer Side
- **Product Listings with Filters:** Sort and search for sarees based on price, category, and latest trends.
- **Multiple Images & Videos:** Each product can have multiple images and videos for a detailed view.
- **Contact Us Form:** Customers can submit inquiries directly through the website.
- **Blog Section:** Stay updated with fashion trends, styling tips, and Kashvi Creation’s latest news.
- **Real-time Order Tracking:** Customers can track the status of their orders live.
- **Customer Reviews:** View product reviews from other buyers; top reviews are highlighted on the homepage.
- **Personalized Chatbot:** Get instant responses to queries related to products, orders, and store details.

### Admin Dashboard
- **Product Management:** Add, edit, or delete products, along with images and videos.
- **Customer Inquiries:** All submitted inquiries are displayed on the admin panel for easy follow-up.
- **Order Management:** Track and update the status of orders in real time.
- **Bulk Promotional Messages:** Send new offers and discounts to all registered customers with a single click.
- **Invoice Generation:** Automatic invoice creation when an order is placed.

## Tech Stack
- **Frontend:** React.js (with Vite for fast development and optimization)
- **Backend:** Node.js with Express.js
- **Database:** MongoDB for storing products, customers, orders, and inquiries
- **Authentication:** Secure user authentication and admin access
- **Chatbot:** Custom AI-powered chatbot designed specifically for Kashvi Creation

## Installation and Setup

### Prerequisites
Ensure you have the following installed on your machine:
- [Node.js](https://nodejs.org/) (LTS version recommended)
- npm (comes with Node.js)

### Steps to Run the Project
1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/your-repo.git
   ```
2. Navigate to the project directory:
   ```bash
   cd your-repo
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
4. Start the development server:
   ```bash
   npm run dev
   ```
